Куда я попал и для чего это нужно?

На основе биоэлектрических потенциалов мышц человека можно управлять физическими объектами. Для этого был создан комплекс управления курсором мыши при помощи ЭМГ.
Электромиография является безвредной и безболезненной. Поэтому, никаких ограничений на длительности и частоту процедур не накладывается. Этот факт и являлся ключевым в выборе темы и проектировании данного устройства.

Как это работает?

1.Собрано устройство и разработано программное обеспечение для него, позволяющее фиксировать изменения потенциалов на поверхности кожи

2. Разработан драйвер для управления курсором компьютерной мыши посредсвом данных, полученных с устройства

3. Третья часть включается в себя защиту данных пользователей системы с помощью технических средств защиты авторских прав DRM, работу с базой данных MongoDB, предоставление API. Само приложение написано на Node.js и AngularJS
